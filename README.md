# Minecraft Data Migrator

## Features
- Swap UUID and Inventory between .dat files.
- Minecraft-themed UI with custom CSS.
- Skin previews via Crafatar.
- Real-time NBT parsing.

## Environment Variables
- `GEMINI_API_KEY`: For any AI-powered assistance (if needed).
- `APP_URL`: Base URL.
